# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |

Version numbers follow `pyproject.toml`.

## Reporting a vulnerability

Report security issues privately to the maintainers. Include:

- Affected component (e.g. `indw.ingest`, `indw.filter`)
- Reproduction steps
- Impact assessment

Do not file public issues for undisclosed vulnerabilities.

## Scope

This policy covers the INDW Python package, CLI, and bundled scripts.
